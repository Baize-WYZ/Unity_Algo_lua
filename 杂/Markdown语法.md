> vscode中预览Markdown、LaTex内容 ctrl+shift+v

1.**声明标题** 使用 # 符号表示标题，# 的数量对应标题的级别（需要文本中单行顶格）    
例如
# 一级标题    
## 二级标题

2.**强调**  粗体强调 ** 或者 __ ，斜体强调 * 或者 _ ，粗斜体强调 *** 或者 ___     
例如   
**粗体**，__粗体__   
*斜体*，_斜体_   
***粗斜体***，___粗斜体___   

3.**换行** 单行末尾多个空格、两次回车、使用html语言中的`<br>`标签来实现换行

4.**显示html标签或者关键字符** 使用反引符号 ```   
例如  `<br>` 或者 `*`

5.**无序列表**: 使用`-`、`*`或`+`作为项目符号    
例如
+ 11
+ 222
+ 33

* 111
* 222
* 333

- 111
- 222
- 333

6.**有序列表**：使用数字加上`.` 例如   
1. 111
2. 222
3. 333
